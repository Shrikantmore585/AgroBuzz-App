# 🌾 AgroBuzz App

## 📌 Project Overview
AgroBuzz is a mobile application developed to help farmers access important agricultural services and market opportunities through a digital platform. The main objective of this application is to reduce the dependency of farmers on middlemen and allow them to sell their agricultural products directly to customers and vendors.

The application also provides useful information such as government schemes and weather updates to help farmers make better decisions in their farming activities.

This project was developed as a **Diploma Final Year Project** using **Flutter and Firebase**.

---

## 🚀 Features
- Farmer product listing
- Direct selling between farmers and buyers
- Government schemes information with official website redirection
- Weather forecasting
- User registration and login using Firebase
- User profile management
- Location-based services

---

## 🛠 Technologies Used
- Flutter (Dart)
- Firebase Authentication
- Firebase Realtime Database / Firestore
- REST API (Weather Data)
- Android Studio
- Visual Studio Code

---

## 📱 Application Modules
1. User Registration & Login  
2. Product Listing Module  
3. Government Schemes Module  
4. Weather Forecast Module  
5. User Profile Management  

---

## 🎯 Objective
The main objective of the AgroBuzz application is to create a digital platform where farmers can easily access agricultural information and sell their products directly in the market, helping them increase their profit and improve transparency in the agricultural supply chain.
